CSV Data Schema

Field Description Example
ioc_type Type of indicator hash-md5, ip, domain, url, etc
ioc_value The actual indicator 192.168.1.1, example.com
threat_name Name of threat/malware Mirai, AgentTesla, Emotet
threat_type Category of threat malware, apt, ransomware
severity Risk level critical, high, medium, low
confidence Confidence in indicator 0-100
first_seen When indicator was first detected 2026-02-15
last_seen When indicator was last active 2026-02-20
mitre_attack MITRE ATT&CK technique ID T1059.001 (Command and Scripting)
tags Additional classification tags botnet, c2, trojan, steal
source Data source attribution public-feed, research, community
description Context and details Command & Control server for Mirai botnet
